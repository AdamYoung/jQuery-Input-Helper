jQuery Input Helper

The input helper is used to set a default value to an input when there is no user entered value in it. The default value is styled differently and will dissapear when the input gets focus.
The alt attribute on <input> element is used to set the default value.
All that is needed is to include the scripts and set the styling for the input when displying the help.

<script type="text/javascript" src="js/jquery-1.4.2.min.js"></script> <script type="text/javascript" src="js/jquery.inputHelper.js"></script> <style> .inputHelp{ color: #666; font-style:italic; } </style>
